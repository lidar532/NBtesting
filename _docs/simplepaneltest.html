Simple Holoviz Panel Test

Simple test to check basic panel functionality.

Import the panel package as pn

import panel as pn

Next, call the extension function so we can display the widgets in the notebook output.

pn.extension()

Button Test.

Now, we define a simple function named clicked(event) to respond each time the button is pushed. The function simply reports the number of times the button was pushed. The widget takes the argument event object.clicks and creates a formatted string which it assignes to the t widgets value parameter.

def clicked( event):
    t.value = f'clicked {event.obj.clicks} times'

Now, we create a button widget named b, a StaticText widget named t. We assign the b button widget to call the clicked(event) function each time the button is pushed.

b = pn.widgets.Button(name='Push me')
t = pn.widgets.StaticText(name="Text:", value="none yet")
watcher = b.on_click( clicked )

Now, we put the b Button and t widgets in a row together and name the row object w. Once the row object is created it can be displayed within Jupyter with the w.servable() when put in the first column on the last row.

w = pn.Row(b,t)
w.servable()

We can also just assign a new string value to the StaticText value t.value directly by executing the following cell:

t.value = 'Hello World XXXX !!'
t

Note the Text widgets value changes in two cells above.
