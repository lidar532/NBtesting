TOGGLE_BUTTON

 TOGGLE_BUTTON (initial_state=0, id='', on_name='Data is On',
                off_name='Data is Off', on_color='primary',
                off_color='danger', change_cb=None)

A panel toggle button with customizable properties. The button’s name and color is set according to the state of the button. If a user call back change_cb is given, it will be called each time the button is toggled. A list will be passed containing the id and the state of the button.

Type Default Details
initial_state int 0 The initial button state upon creation.
id str An identifier string.
on_name str Data is On Message displayed on the button when in ON state
off_name str Data is Off Message displayed on button when in Off state
on_color str primary Color when ON
off_color str danger Color when Off
change_cb NoneType None An optional user call back function called when toggled.
